# mhe_estimator
Mobile robot Moving Horizon estimator (MHE)

# External dependencies
    ##  Ackermann_msg:
    *` sudo apt-get install ros-melodic-ackermann-msgs`
    ##  CASADI:
	* `cd ~/` 	(home)
	* `git clone https://github.com/casadi/casadi.git`
	* `cd ~/casadi/`
	* `mkdir build`
	* `cd build`
	* `cmake ..`
	* `make`
	* `sudo make install`
